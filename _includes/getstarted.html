Get Started


1、git clone

# git clone https://github.com/TrustNoteDevelopers/RPC.git

2、install

# cd RPC
# npm install

3、init

# npm run init

4、run

# npm run rpc

python-sdk API:

1、git clone

# git clone https://github.com/TrustNoteDevelopers/python-sdk.git

2、run RESTful API

cd python-sdk && python web.py

now open http://localhost

get all address:

http://localhost/api/address/all

make a new address:

http://localhost/api/address/new

search balance of address

http://localhost/api/balance/VOXKRZWYYUP6JSSY7FZRKCQT3FAPWCFJ

pay:(POST method)

http://localhost/api/pay
$.post( "/api/pay", {
                    address:"5ZCKATEOSYWIEXSOSIK2ZVMDXHTO6F32",
                    amount:5,
                    token:"kPI5sZc1e7vG/nik67qDP4N8sjAnnhYRsUTUB/YvsTY="
                },function(response){
                    console.log(response)
                    response=JSON.parse(response)
                    if(response.hasOwnProperty("error")){
                        alert('抱歉，兑换失败')
                    }else{
                        alert('申请兑换成功')
                    }
                });
